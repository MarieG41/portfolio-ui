05 || Profil utilisateur pour une application de rencontre canine

Ce projet est un profil utilisateur pour une application de rencontre web et mobile pour les animaux et leurs maîtres.
Cible: les propriétaire et passionnés d'animaux

L'interface est simple, user-friendly et facile à comprendre et à prendre en main. Les couleurs sont neutres sauf pour ce jaune qui permet diriger le regard et d'indiquer aux utilisateurs les éléments cliquables.
